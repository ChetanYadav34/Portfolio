# Chetan Yadav - Personal Portfolio

A modern, responsive personal portfolio built with Next.js, Tailwind CSS, and Framer Motion.
Features a dark theme with gold accent highlights and GitHub project integration.

## Local Development

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploying to Vercel

The easiest way to deploy this Next.js app is to use the [Vercel Platform](https://vercel.com/new).

### Steps:
1. **Push your code to GitHub:** Create a new repository on GitHub and push your local portfolio code to it.
2. **Import Project to Vercel:** Go to Vercel (https://vercel.com), log in with your GitHub account, and click **"Add New..." > "Project"**.
3. **Select Repository:** Choose the repository you just created.
4. **Configure Project:** Vercel will automatically detect that it's a Next.js project. You don't need to change any build settings.
5. **Deploy:** Click **"Deploy"**. Your portfolio will be live in a few minutes, complete with continuous deployment out of the box!
